3. **Phases**: Organize work into numbered phases with individual markdown files
4. **Research**: Optional research subdirectory for analysis
5. **Reporting**: Automatic reports generated to `plans/reports/` by agents

1. **Source**: `src/content/docs/` (English) or `src/content/docs-vi/` (Vietnamese)

3. **Plan**: Create plan if changes are substantial

5. **Quality**: Verify build passes with `bun run build`
