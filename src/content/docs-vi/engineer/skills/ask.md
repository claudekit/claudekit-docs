# `ck:ask`

Trả lời các câu hỏi kỹ thuật và kiến trúc về codebase của bạn. Skill này cung cấp câu trả lời có nghiên cứu, chuyên sâu thay vì những phản hồi hời hợt.

## Sử Dụng

```
/ck:ask [câu hỏi của bạn]
```

## Ví Dụ

```bash
/ck:ask how is authentication currently implemented?
/ck:ask what's the database schema for orders?
/ck:ask explain the middleware chain in this project
```

## Skill Này Làm Gì

- Đọc các code files liên quan để tìm câu trả lời
- Cung cấp responses có nhận thức về context dựa trên codebase thực tế của bạn
- Tham chiếu đến các files và số dòng cụ thể
- Giải thích các quyết định kiến trúc và patterns được tìm thấy trong code của bạn
