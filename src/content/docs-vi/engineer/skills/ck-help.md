# Help

Hướng dẫn sử dụng ClaudeKit và khám phá skills. Giúp bạn tìm đúng skill cho mọi tác vụ và hiểu cách sử dụng ClaudeKit hiệu quả.

## Sử Dụng

```
/ck:ck-help [chủ đề hoặc câu hỏi]
```

## Ví Dụ

```bash
/ck:ck-help                    # Trợ giúp tổng quát
/ck:ck-help how to plan        # Tìm planning skills
/ck:ck-help git workflow        # Các skills liên quan đến Git
```

## Skill Này Làm Gì

- Tạo danh mục skills từ các skills đã cài đặt
- Định tuyến queries đến tài liệu skill phù hợp
- Cung cấp usage examples và workflow guidance
- Xử lý subcommand discovery cho các skills phức tạp
