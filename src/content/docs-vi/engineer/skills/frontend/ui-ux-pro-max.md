# UI/UX Pro Max Skill

Truy cập ngay lập tức 50 UI styles, 21 color palettes, 50 font pairings và 20 chart types—tất cả được tùy chỉnh cho loại sản phẩm và tech stack của bạn.

```

"Xây dựng landing page cho spa làm đẹp của tôi"
"Hiệu ứng hover navbar cảm thấy janky, sửa nó"
```

Claude tự động kích hoạt skill này khi bạn làm việc với các tác vụ UI/UX.

- **Style guide** với màu sắc, hiệu ứng và đề xuất framework

## Quy Tắc Chất Lượng Claude Tuân Theo

### Icons & Visual Elements
- Chỉ SVG icons (Heroicons, Lucide)—không emojis
- Hover states ổn định với color/opacity transitions
- Verified brand logos từ Simple Icons

### Interaction
- `cursor-pointer` trên tất cả clickable elements
- `transition-colors duration-200` cho smooth feedback
- Visible focus states cho keyboard navigation

### Light/Dark Mode
- Glass cards: `bg-white/80` hoặc cao hơn trong light mode
- Text contrast: `#0F172A` (slate-900) cho body
- Borders: `border-gray-200` visible trong light mode

### Layout
- Floating navbar với `top-4 left-4 right-4` spacing
- Responsive tại 320px, 768px, 1024px, 1440px
- Không horizontal scroll trên mobile

## Pre-Delivery Checklist
