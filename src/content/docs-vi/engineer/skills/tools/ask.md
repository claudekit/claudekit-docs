Trả lời các câu hỏi kỹ thuật và kiến trúc về codebase của bạn. Skill này cung cấp câu trả lời chính xác dựa trên nghiên cứu thực tế thay vì phản hồi hời hợt.

## Cách Dùng

```
/ck:ask [câu hỏi của bạn]
```

## Ví Dụ

```bash
/ck:ask how is authentication currently implemented?
/ck:ask what's the database schema for orders?
/ck:ask explain the middleware chain in this project
```

## Chức Năng

- Đọc các file code liên quan để tìm câu trả lời
- Cung cấp phản hồi có nhận thức ngữ cảnh dựa trên codebase thực tế của bạn
- Tham chiếu đến các file và số dòng cụ thể
- Giải thích các quyết định kiến trúc và các pattern được tìm thấy trong code của bạn
