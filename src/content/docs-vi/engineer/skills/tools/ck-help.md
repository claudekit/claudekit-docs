Hướng dẫn sử dụng ClaudeKit và khám phá skill. Giúp bạn tìm đúng skill cho mọi tác vụ và hiểu cách sử dụng ClaudeKit hiệu quả.

## Cách Dùng

```
/ck:ck-help [chủ đề hoặc câu hỏi]
```

## Ví Dụ

```bash
/ck:ck-help                    # Trợ giúp chung
/ck:ck-help how to plan        # Tìm skill lập kế hoạch
/ck:ck-help git workflow        # Skill liên quan đến Git
```

## Chức Năng

- Tạo danh mục skill từ các skill đã cài đặt
- Định tuyến câu hỏi đến tài liệu skill phù hợp
- Cung cấp ví dụ sử dụng và hướng dẫn quy trình
- Xử lý khám phá lệnh con cho các skill phức tạp
