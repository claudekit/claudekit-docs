Viết nhật ký phát triển để ghi lại các quyết định kỹ thuật, thách thức, và tiến độ. Hữu ích để duy trì hồ sơ về hành trình phát triển của bạn.

## Cách Dùng

```
/ck:journal [chủ đề hoặc mô tả bài nhập]
```

## Ví Dụ

```bash
/ck:journal debugging the auth token refresh loop
/ck:journal decided to switch from REST to GraphQL
/ck:journal completed phase 2 of the migration
```
