Sử dụng các công cụ từ máy chủ Model Context Protocol (MCP) được cấu hình trong dự án của bạn. Khám phá và thực thi các khả năng MCP.

## Cách Dùng

```
/ck:use-mcp [mô tả tác vụ]
```

## Chức Năng

- Khám phá các công cụ MCP có sẵn từ các máy chủ đã cấu hình
- Chọn công cụ phù hợp nhất cho tác vụ của bạn
- Thực thi các công cụ MCP với các tham số phù hợp
- Trả về kết quả theo định dạng dễ đọc
