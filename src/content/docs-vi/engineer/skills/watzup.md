# Watzup

Xem lại các thay đổi gần đây và kết thúc phiên làm việc hiện tại. Tạo bản tóm tắt những gì đã hoàn thành và bước tiếp theo.

## Cách Dùng

```
/ck:watzup
```

## Chức Năng

- Xem lại git log để tìm các commits gần đây
- Tóm tắt các thay đổi đã thực hiện trong phiên hiện tại
- Liệt kê các file đã chỉnh sửa và mục đích của chúng
- Gợi ý các bước tiếp theo
