# SEO Specialist

> **Người bảo vệ khả năng hiển thị tìm kiếm của bạn** - Sửa vấn đề kỹ thuật, tối ưu nội dung và leo xếp hạng

## Agent Này Làm Gì

Site của bạn nhanh và nội dung của bạn tốt, nhưng bạn bị kẹt ở trang 2 của Google. Bạn đã nghe về Core Web Vitals và schema markup nhưng không biết bắt đầu từ đâu. Đối thủ của bạn xếp hạng cao hơn mặc dù sản phẩm của bạn tốt hơn.

**Vấn Đề**: SEO trở nên cực kỳ phức tạp. Đây không chỉ về từ khóa nữa - đó là hiệu suất kỹ thuật, trải nghiệm mobile, dữ liệu có cấu trúc, tín hiệu E-E-A-T và hàng trăm yếu tố xếp hạng khác. Hầu hết các đội thiếu chuyên môn để kiểm tra và tối ưu tất cả những yếu tố này.

**Giải Pháp**: SEO Specialist là chuyên gia SEO kỹ thuật senior của bạn, người biết mọi yếu tố xếp hạng quan trọng. Agent này thực hiện kiểm tra SEO kỹ thuật toàn diện, tối ưu nội dung on-page cho ý định tìm kiếm, tạo JSON-LD schema markup đúng, phân tích Core Web Vitals, phát triển chiến lược xây dựng link và xác định chính xác cái gì đang giữ xếp hạng của bạn lại.

Agent này tuân theo hướng dẫn mới nhất của Google và sử dụng kỹ thuật tối ưu đã được chứng minh từ các site xếp hạng cao nhất để cải thiện khả năng hiển thị tìm kiếm của bạn.

## Bắt Đầu Nhanh

Chạy kiểm tra SEO đầu tiên trong 30 giây:

```bash
# Kiểm tra site của bạn về vấn đề SEO
/ask "Thực hiện kiểm tra SEO kỹ thuật trên [your-domain.com]"

# Agent sẽ phân tích khả năng crawl, indexation, tốc độ, trải nghiệm mobile và nhiều hơn
```

## Khả Năng

### Kiểm Tra SEO Kỹ Thuật

Nhận kiểm tra sức khỏe toàn diện của nền tảng kỹ thuật site của bạn. Agent crawl cấu trúc site của bạn, kiểm tra trạng thái indexation, phân tích số liệu tốc độ trang, review tối ưu hóa mobile, đánh giá Core Web Vitals (LCP, FID, CLS) và review triển khai schema markup. Bạn sẽ nhận danh sách vấn đề ưu tiên theo mức độ nghiêm trọng.

**Ví dụ**: Agent xác định 23 vấn đề quan trọng bao gồm thiếu canonical tag, tải ảnh chậm (LCP 4.2s) và lỗi crawl ngăn Google index 15% trang của bạn.

### Tối Ưu Hóa Nội Dung

Cải thiện yếu tố SEO on-page của mỗi trang. Agent phân tích title tag và meta description, review cấu trúc header (H1-H6), kiểm tra cấu trúc liên kết nội bộ, đánh giá chất lượng và độ sâu nội dung, và đánh giá tín hiệu E-E-A-T (Experience, Expertise, Authoritativeness, Trust). Bạn sẽ nhận đề xuất cụ thể cho mỗi trang.

### Phân Tích Từ Khóa

Hiểu bạn nên xếp hạng cho cái gì và tại sao bạn không. Agent thực hiện nghiên cứu từ khóa với dữ liệu khối lượng tìm kiếm, ánh xạ ý định tìm kiếm sang loại nội dung, tính điểm độ khó từ khóa, phân tích tính năng SERP và đối thủ, và xác định cơ hội low-hanging fruit.

### Chiến Lược Xây Dựng Link

Xây dựng thẩm quyền thông qua backlink chiến lược. Agent phân tích profile backlink hiện tại của bạn, xác định cơ hội khoảng trống link, đề xuất chiến lược outreach cho site liên quan, tìm cơ hội guest posting trong niche của bạn và phát triển ý tưởng PR kỹ thuật số để kiếm link tự nhiên.

### Tạo JSON-LD Schema

Giúp Google hiểu nội dung của bạn với dữ liệu có cấu trúc đúng. Agent tạo schema markup cho article, product, organization, FAQ, how-to, local business và review. Schema đúng có thể kiếm rich snippet, tăng click-through rate 20-30%.

### Phân Tích SEO Đối Thủ

Xem chính xác nơi đối thủ đang đánh bại bạn. Agent xác định khoảng trống từ khóa cạnh tranh, so sánh profile backlink, phân tích khoảng trống nội dung trong phạm vi của bạn, theo dõi quyền sở hữu tính năng SERP và trích xuất thông tin chiến lược từ cái gì hoạt động cho họ.

## Khi Nào Dùng

Sử dụng SEO Specialist khi bạn cần:
- Kiểm tra site của bạn về vấn đề SEO kỹ thuật ngăn xếp hạng
- Tối ưu nội dung hiện tại để cải thiện vị trí tìm kiếm
- Tạo schema markup đúng cho rich snippet
- Phân tích tại sao đối thủ xếp hạng cao hơn bạn
- Phát triển chiến lược xây dựng link cho xây dựng thẩm quyền
- Sửa vấn đề Core Web Vitals ảnh hưởng trải nghiệm người dùng

## Quy Trình Ví Dụ

### Quy Trình 1: Kiểm Tra Kỹ Thuật Hoàn Chỉnh

Xác định và sửa vấn đề kỹ thuật:

```bash
# Bước 1: Chạy kiểm tra toàn diện
/ask "Kiểm tra [your-domain.com] về vấn đề SEO kỹ thuật"

# Agent phân tích:
# - Khả năng crawl và indexation
# - Tốc độ trang và Core Web Vitals
# - Tối ưu hóa mobile
# - Schema markup
# - Liên kết nội bộ
# - XML sitemap và robots.txt

# Bước 2: Nhận kế hoạch hành động ưu tiên
# Agent trả về vấn đề xếp hạng theo:
# - Mức độ nghiêm trọng (cao/trung bình/thấp)
# - Trang bị ảnh hưởng (số lượng)
# - Độ khó sửa (dễ/trung bình/khó)
# - Tác động dự kiến lên xếp hạng
```

**Đầu ra**: Báo cáo kiểm tra kỹ thuật với sửa chữa ưu tiên và hướng dẫn triển khai.

### Quy Trình 2: Tối Ưu Hóa Nội Dung

Tối ưu trang cho từ khóa mục tiêu:

```bash
# Cung cấp URL và từ khóa mục tiêu
/ask "Tối ưu [page-url] cho từ khóa '[từ khóa mục tiêu]'"

# Agent sẽ:
# - Phân tích yếu tố xếp hạng hiện tại
# - Review title, meta, header
# - Đánh giá độ sâu nội dung vs. đối thủ
# - Kiểm tra liên kết nội bộ
# - Cung cấp đề xuất tối ưu cụ thể
```

**Đầu ra**: Checklist tối ưu cụ thể trang với đề xuất trước/sau.

### Quy Trình 3: Tạo Schema Markup

Thêm dữ liệu có cấu trúc cho rich snippet:

```bash
# Yêu cầu schema cho loại nội dung của bạn
/ask "Tạo JSON-LD schema cho trang [article/product/FAQ]"

# Agent tạo:
# - Code JSON-LD định dạng đúng
# - Tất cả field bắt buộc và đề xuất
# - Hướng dẫn triển khai
# - Đề xuất testing
```

**Đầu ra**: Code JSON-LD sẵn sàng production để paste vào trang của bạn.

### Quy Trình 4: Phân Tích Khoảng Trống Đối Thủ

Tìm cơ hội đối thủ đang khai thác:

```bash
# So sánh với đối thủ
/ask "Phân tích khoảng trống SEO giữa chúng tôi và [competitor-domain]"

# Agent xác định:
# - Từ khóa họ xếp hạng mà bạn không
# - Backlink họ có mà bạn không
# - Chủ đề nội dung họ đề cập mà bạn đang thiếu
# - Tính năng SERP họ sở hữu
# - Cơ hội traffic ước tính
```

**Đầu ra**: Phân tích khoảng trống đối thủ với cơ hội khả thi.

## Thực Hành Tốt Nhất

1. **Sửa Vấn Đề Quan Trọng Trước**: Tập trung vào vấn đề kỹ thuật mức độ nghiêm trọng cao ảnh hưởng crawling, indexing hoặc trải nghiệm người dùng. Những cái này có tác động lớn nhất lên xếp hạng.

2. **Mobile First**: Google sử dụng mobile-first indexing. Đảm bảo trải nghiệm mobile của bạn xuất sắc - test mọi thứ trên thiết bị mobile.

3. **Tốc Độ Trang Quan Trọng**: Core Web Vitals là yếu tố xếp hạng. Nhắm LCP < 2.5s, FID < 100ms, CLS < 0.1. Agent sẽ gắn cờ vấn đề hiệu suất.

4. **Chất Lượng Hơn Số Lượng**: Một trang tối ưu tốt đánh bại mười trang tầm thường. Sử dụng hướng dẫn tối ưu nội dung của agent để tạo nội dung toàn diện, hữu ích.

5. **Schema Là Bất Động Sản Miễn Phí**: Schema markup đúng có thể kiếm rich snippet, tăng CTR 20-30%. Triển khai nó trên tất cả trang đủ tiêu chuẩn.

6. **Theo Dõi Đối Thủ**: SEO có tính cạnh tranh. Sử dụng phân tích đối thủ của agent hàng quý để dẫn đầu thay đổi trong thị trường của bạn.

## Tích Hợp MCP

SEO Specialist có thể tích hợp với công cụ bên ngoài:

- **ReviewWebsite MCP**: Scrape URL trực tiếp, nghiên cứu từ khóa, phân tích backlink
- **Google Search Console MCP**: Dữ liệu hiệu suất tìm kiếm thực, trạng thái indexing, xếp hạng query

Khi MCP server không khả dụng, agent sử dụng phương pháp phân tích thủ công.
