# ckm:analyze

> Biến dữ liệu marketing thành insights hành động với báo cáo tự động và trực quan hóa.

## Kỹ Năng Này Làm Gì

Skill `ckm:analyze` giúp bạn phân tích hiệu suất marketing từ nhiều nguồn dữ liệu khác nhau — GA4, nền tảng quảng cáo, email, mạng xã hội — và tổng hợp thành báo cáo rõ ràng, có thể hành động ngay.

Thay vì mất hàng giờ kéo dữ liệu thủ công, skill này tự động hóa việc thu thập, tính toán chỉ số KPI và tạo báo cáo Markdown với biểu đồ Mermaid.js kèm khuyến nghị ưu tiên.

## Bắt Đầu Nhanh

```
/ckm:analyze
```

Hoặc kèm ngữ cảnh cụ thể:

```
/ckm:analyze Phân tích hiệu suất chiến dịch Q1 và so sánh với Q4 năm ngoái
```

## Tính Năng Chính

- **Tổng hợp dữ liệu đa kênh**: GA4, quảng cáo, email, mạng xã hội
- **Tính toán KPI tự động**: CAC, ROAS, CVR, LTV, churn rate
- **Báo cáo Markdown**: Bảng hiệu suất + biểu đồ xu hướng Mermaid.js
- **Khuyến nghị hành động**: 3-5 ưu tiên rõ ràng với người chịu trách nhiệm
- **So sánh kỳ trước**: Theo dõi tăng trưởng và phát hiện bất thường

## Ví Dụ Sử Dụng

**Báo cáo hiệu suất hàng tuần:**
```
/ckm:analyze Tạo báo cáo hiệu suất tuần này so với tuần trước, tập trung vào CAC và ROAS
```

**Phân tích chiến dịch:**
```
/ckm:analyze Đánh giá chiến dịch email tháng 3 — tỷ lệ mở, CTR, chuyển đổi
```

**Phân tích kênh:**
```
/ckm:analyze So sánh hiệu quả các kênh thu hút: paid, organic, referral
```
