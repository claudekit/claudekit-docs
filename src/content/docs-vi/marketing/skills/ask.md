# ckm:ask

> Nhận câu trả lời chuyên sâu cho các câu hỏi kỹ thuật và kiến trúc của hệ thống marketing.

## Kỹ Năng Này Làm Gì

Skill `ckm:ask` là cửa ngõ để hỏi bất kỳ câu hỏi nào liên quan đến kỹ thuật, kiến trúc hệ thống hoặc cách hoạt động của ClaudeKit Marketing. Từ câu hỏi về cấu hình script đến thiết kế workflow phức tạp.

Thay vì đọc tài liệu phân tán hoặc thử nghiệm mò, skill này tra cứu ngữ cảnh dự án và cung cấp câu trả lời chính xác, có ví dụ thực tế.

## Bắt Đầu Nhanh

```
/ckm:ask Làm thế nào để cấu hình GA4 API?
```

```
/ckm:ask Script nào dùng để tạo ảnh với Imagen 4?
```

## Tính Năng Chính

- **Tra cứu tài liệu thông minh**: Tìm câu trả lời trong references, scripts, CLAUDE.md
- **Giải thích kiến trúc**: Mô tả cách các thành phần hoạt động cùng nhau
- **Hướng dẫn cấu hình**: Bước-bước cho thiết lập kỹ thuật
- **Debug hỗ trợ**: Phân tích lỗi và đề xuất giải pháp
- **Ví dụ code**: Cung cấp đoạn code sẵn dùng cho tác vụ cụ thể

## Ví Dụ Sử Dụng

**Câu hỏi cấu hình:**
```
/ckm:ask Biến môi trường nào cần thiết cho tích hợp Google Search Console?
```

**Câu hỏi về workflow:**
```
/ckm:ask Workflow nào phù hợp nhất cho chiến dịch email drip?
```

**Debug:**
```
/ckm:ask Script gemini_batch_process.py báo lỗi 429, cách xử lý?
```
