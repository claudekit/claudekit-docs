# ckm:ck-help

> Hướng dẫn đầy đủ về cách sử dụng ClaudeKit Marketing — skills, agents, workflows và best practices.

## Kỹ Năng Này Làm Gì

Skill `ckm:ck-help` là hệ thống trợ giúp tích hợp của ClaudeKit Marketing. Cung cấp hướng dẫn sử dụng cho bất kỳ skill hoặc agent nào, giải thích workflow, gợi ý skill phù hợp cho từng tác vụ và giúp bạn khám phá toàn bộ khả năng của hệ thống.

Đây là điểm bắt đầu tốt nhất khi bạn không chắc nên dùng skill nào cho nhu cầu của mình.

## Bắt Đầu Nhanh

```
/ckm:ck-help
```

Hỏi về skill cụ thể:

```
/ckm:ck-help ckm:email Hướng dẫn chi tiết về skill email
```

## Tính Năng Chính

- **Skill discovery**: Tìm skill phù hợp nhất cho tác vụ của bạn
- **Quick reference**: Tóm tắt cú pháp và options cho mọi skill
- **Workflow guides**: Hướng dẫn workflow kết hợp nhiều skills
- **Troubleshooting**: Chẩn đoán và giải quyết vấn đề thường gặp
- **Best practices**: Gợi ý cách dùng hiệu quả nhất

## Ví Dụ Sử Dụng

**Tìm skill phù hợp:**
```
/ckm:ck-help Tôi cần tạo chiến dịch email drip — nên dùng skill nào?
```

**Hỏi về workflow:**
```
/ckm:ck-help Workflow tốt nhất để ra mắt sản phẩm mới với ClaudeKit Marketing là gì?
```

**Xem tất cả skills:**
```
/ckm:ck-help list Liệt kê tất cả skills và agents có sẵn với mô tả ngắn
```

**Khắc phục sự cố:**
```
/ckm:ck-help troubleshoot Script Gemini API báo lỗi 429, cách xử lý?
```
