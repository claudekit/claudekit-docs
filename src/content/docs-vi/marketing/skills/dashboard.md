# ckm:dashboard

> Khởi chạy Marketing Dashboard để theo dõi KPI và hiệu suất chiến dịch theo thời gian thực.

## Kỹ Năng Này Làm Gì

Skill `ckm:dashboard` khởi tạo và quản lý Marketing Dashboard — giao diện trực quan tổng hợp dữ liệu từ tất cả kênh marketing vào một nơi. Bao gồm biểu đồ xu hướng, cảnh báo bất thường và snapshot hiệu suất hàng ngày/tuần/tháng.

Dashboard chạy cục bộ và cập nhật tự động từ các nguồn dữ liệu đã cấu hình.

## Bắt Đầu Nhanh

```
/ckm:dashboard
```

Để mở dashboard hiện có:

```
/ckm:dashboard open
```

## Tính Năng Chính

- **Tổng quan KPI**: Tất cả chỉ số quan trọng trên một màn hình
- **Biểu đồ xu hướng**: Mermaid.js visualization cho dữ liệu theo thời gian
- **Cảnh báo bất thường**: Thông báo khi chỉ số vượt ngưỡng
- **Báo cáo tự động**: Snapshot hàng ngày/tuần/tháng
- **Đa kênh**: GA4, email, mạng xã hội, quảng cáo trong cùng giao diện

## Ví Dụ Sử Dụng

**Khởi chạy dashboard mới:**
```
/ckm:dashboard Tạo dashboard theo dõi CAC, ROAS, CVR cho Q2 2025
```

**Cập nhật cấu hình:**
```
/ckm:dashboard Thêm theo dõi chỉ số email vào dashboard hiện tại
```

**Tạo báo cáo snapshot:**
```
/ckm:dashboard Tạo snapshot hiệu suất tuần này để chia sẻ với team
```
