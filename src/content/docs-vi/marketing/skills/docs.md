# ckm:docs

> Quản lý tài liệu dự án marketing — từ khởi tạo đến cập nhật và tóm tắt nhanh.

## Kỹ Năng Này Làm Gì

Skill `ckm:docs` giúp bạn duy trì tài liệu dự án marketing luôn cập nhật và có tổ chức. Tạo cấu trúc tài liệu chuẩn, cập nhật sau mỗi sprint, và tóm tắt trạng thái dự án cho các bên liên quan.

Tài liệu tốt là nền tảng để AI agents hoạt động hiệu quả — `ckm:docs` đảm bảo ngữ cảnh dự án luôn chính xác và đầy đủ.

## Bắt Đầu Nhanh

```
/ckm:docs
```

Khởi tạo tài liệu cho dự án mới:

```
/ckm:docs init
```

## Tính Năng Chính

- **Khởi tạo cấu trúc docs**: Tạo thư mục `docs/` với các file chuẩn
- **Cập nhật tự động**: Đồng bộ docs sau khi hoàn thành tính năng
- **Tóm tắt dự án**: Tạo executive summary cho stakeholders
- **Changelog**: Ghi lại lịch sử thay đổi tự động
- **Context injection**: Đưa ngữ cảnh docs vào system prompt cho agents

## Ví Dụ Sử Dụng

**Khởi tạo docs mới:**
```
/ckm:docs init Tạo cấu trúc tài liệu cho dự án marketing automation Q3
```

**Cập nhật sau sprint:**
```
/ckm:docs update Cập nhật docs sau khi hoàn thành tích hợp GA4 và email automation
```

**Tóm tắt dự án:**
```
/ckm:docs summary Tóm tắt trạng thái dự án hiện tại để báo cáo cho ban lãnh đạo
```
