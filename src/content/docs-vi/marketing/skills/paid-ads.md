# Paid Ads

> Tối đa hóa ROI quảng cáo trả phí thông qua chiến lược nhắm mục tiêu chính xác và tối ưu liên tục.

## Kỹ Năng Này Làm Gì

Skill `ckm:paid-ads` cung cấp framework toàn diện cho quảng cáo trả phí đa nền tảng. Từ nghiên cứu đối tượng, cấu trúc chiến dịch, viết copy quảng cáo đến phân tích hiệu suất và phân bổ ngân sách tối ưu.

## Bắt Đầu Nhanh

```
/ckm:paid-ads
```

Hoặc chỉ định nền tảng và mục tiêu:

```
/ckm:paid-ads Tạo chiến dịch Google Search Ads cho SaaS B2B — mục tiêu trial signups
```

## Nền Tảng Hỗ Trợ

| Nền Tảng | Phù Hợp Nhất | Định Dạng |
|----------|-------------|---------|
| Google Search Ads | Intent cao, B2B/B2C | Text, Shopping |
| Meta (FB/IG) | Awareness, retargeting | Image, Video, Carousel |
| LinkedIn | B2B, professional audience | Sponsored content, InMail |
| TikTok | Gen Z, brand awareness | Short video |

## Tính Năng Chính

- **Cấu trúc chiến dịch**: Campaign → Ad Set → Ad với logic phân cấp
- **Nghiên cứu đối tượng**: Custom audiences, lookalike, interest targeting
- **Copy framework**: Headline, description, CTA theo từng nền tảng
- **Bid strategy**: Manual CPC, tCPA, tROAS — lựa chọn theo giai đoạn
- **Performance analysis**: ROAS, CTR, CPC, CPL — phân tích và tối ưu

## Ví Dụ Sử Dụng

**Thiết kế chiến dịch:**
```
/ckm:paid-ads Thiết kế cấu trúc chiến dịch Meta Ads cho launch sản phẩm SaaS, ngân sách $5K/tháng
```

**Tối ưu đang chạy:**
```
/ckm:paid-ads Phân tích dữ liệu chiến dịch Google Ads này và đề xuất tối ưu — ROAS hiện tại 2.1x
```

**Viết copy quảng cáo:**
```
/ckm:paid-ads Viết 5 biến thể headline Google Search Ads cho từ khóa "project management software"
```

## Checklist Trước Khi Chạy

- [ ] Pixel/Conversion tracking đã thiết lập và kiểm thử
- [ ] Landing page tối ưu cho mục tiêu chuyển đổi
- [ ] Ngân sách test đủ để học (tối thiểu 50 conversions/nhóm)
- [ ] Negative keywords đã thiết lập (Google)
- [ ] Exclusion lists đã cấu hình (Meta)
