# `ckm:pricing-strategy`

> Thiết kế cấu trúc giá tối ưu hóa doanh thu và tỷ lệ chuyển đổi thông qua phân tích giá trị và tâm lý học giá.

## Kỹ Năng Này Làm Gì

Skill `ckm:pricing-strategy` giúp thiết kế và tối ưu chiến lược định giá cho sản phẩm SaaS và dịch vụ. Phân tích giá trị cung cấp, nghiên cứu cạnh tranh, thiết kế tier pricing và đề xuất chiến lược kiếm tiền phù hợp với giai đoạn tăng trưởng.

## Bắt Đầu Nhanh

```
/ckm:pricing-strategy
```

Hoặc mô tả sản phẩm:

```
/ckm:pricing-strategy Thiết kế pricing cho SaaS B2B project management, target mid-market
```

## Các Mô Hình Giá Phổ Biến

| Mô Hình | Phù Hợp | Ví Dụ |
|---------|---------|-------|
| Freemium | Viral growth, PLG | Notion, Figma |
| Flat rate | Simple, all-in-one | Basecamp |
| Per-seat | Team collaboration | Slack, Linear |
| Usage-based | Variable consumption | Twilio, AWS |
| Tiered | Segment differentiation | HubSpot, Intercom |

## Tính Năng Chính

- **Value metric analysis**: Xác định đơn vị giá trị phù hợp nhất
- **Competitive pricing**: Nghiên cứu và định vị so với đối thủ
- **Tier design**: Thiết kế Free/Starter/Pro/Enterprise với tính năng phân tầng
- **Packaging**: Nhóm tính năng tối ưu willingness-to-pay
- **Pricing page**: Khuyến nghị copy và layout trang giá

## Ví Dụ Sử Dụng

**Phân tích pricing hiện tại:**
```
/ckm:pricing-strategy Đánh giá cấu trúc pricing hiện tại — conversion rate thấp ở tier Pro
```

**Thiết kế freemium:**
```
/ckm:pricing-strategy Thiết kế giới hạn freemium: đủ hấp dẫn để dùng nhưng đủ friction để nâng cấp
```

**Pricing page optimization:**
```
/ckm:pricing-strategy Đề xuất cải tiến trang pricing để tăng tỷ lệ chọn tier Pro
```

## Nguyên Tắc Tâm Lý Học Giá

- **Anchoring**: Hiển thị tier đắt nhất trước để làm mốc tham chiếu
- **Decoy pricing**: Tier middle làm cho tier target hấp dẫn hơn
- **Annual discount**: Giảm 20-40% cho gói năm — tăng LTV, giảm churn
- **Charm pricing**: $49 thay vì $50 — hiệu quả ở B2C
