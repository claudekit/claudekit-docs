# `ckm:write`

> Viết copy marketing chuyển đổi cao — từ headline, landing page đến bài blog và long-form content.

## Kỹ Năng Này Làm Gì

Skill `ckm:write` tạo nội dung marketing chất lượng cao với giọng nói thương hiệu nhất quán. Bao gồm copywriting chuyển đổi (headlines, CTAs, landing pages), bài blog SEO-optimized và long-form content như whitepapers hay case studies.

Mỗi nội dung được viết với audience cụ thể, giai đoạn funnel và mục tiêu chuyển đổi rõ ràng.

## Bắt Đầu Nhanh

```
/ckm:write
```

Viết nội dung cụ thể:

```
/ckm:write Viết bài blog 1500 chữ về "5 cách AI thay đổi email marketing" — SEO-optimized, friendly tone
```

## Tính Năng Chính

- **Copywriting CRO**: Headlines, value propositions, CTAs tối ưu chuyển đổi
- **Blog content**: Bài blog SEO-friendly với structure rõ ràng
- **Landing pages**: Copy đầy đủ cho landing page theo AIDA framework
- **Long-form**: Whitepapers, case studies, guides chuyên sâu
- **Brand voice**: Áp dụng brand guidelines tự động

## Ví Dụ Sử Dụng

**Landing page:**
```
/ckm:write landing-page Viết copy landing page cho free trial — headline, benefits, social proof, CTA
```

**Bài blog:**
```
/ckm:write blog "10 Marketing Automation Trends 2025" — 2000 chữ, target keyword "marketing automation trends"
```

**Email copy:**
```
/ckm:write email Viết email re-engagement cho users inactive 60 ngày — empathetic tone, clear CTA
```
