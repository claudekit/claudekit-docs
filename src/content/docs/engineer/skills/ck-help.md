ClaudeKit usage guide and skill discovery. Helps you find the right skill for any task and understand how to use ClaudeKit effectively.

## Usage

```
/ck:ck-help [topic or question]
```

## Examples

```bash
/ck:ck-help                    # General help
/ck:ck-help how to plan        # Find planning skills
/ck:ck-help git workflow        # Git-related skills
```

## What It Does

- Generates a skills catalog from installed skills
- Routes queries to relevant skill documentation
- Provides usage examples and workflow guidance
- Handles subcommand discovery for complex skills
