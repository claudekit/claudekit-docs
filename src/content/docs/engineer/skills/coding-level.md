Set your coding experience level so ClaudeKit tailors its explanations, code comments, and output complexity to match your expertise.

## Usage

```
/coding-level [level]
```

## Levels

| Level | Description |
|-------|-------------|
| beginner | Detailed explanations, more comments, step-by-step guidance |
| intermediate | Balanced explanations, standard code output |
| advanced | Concise output, minimal hand-holding, expert-level patterns |
