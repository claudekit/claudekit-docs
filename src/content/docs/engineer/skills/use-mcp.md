Utilize tools from Model Context Protocol (MCP) servers configured in your project. Discovers and executes MCP capabilities.

## Usage

```
/use-mcp [task description]
```

## What It Does

- Discovers available MCP tools from configured servers
- Selects the most relevant tool for your task
- Executes MCP tools with appropriate parameters
- Returns results in a readable format
