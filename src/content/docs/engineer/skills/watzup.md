# `ck:watzup`

Review recent changes and wrap up your current work session. Generates a summary of what was accomplished and what's next.

## Usage

```
/ck:watzup
```

## What It Does

- Reviews git log for recent commits
- Summarizes changes made in the current session
- Lists modified files and their purposes
- Suggests next steps
