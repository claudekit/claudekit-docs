# `ckm:email`

> Write high-converting emails, build sequences, and design automation flows for every stage of the funnel.

## What This Skill Does

The Email skill generates email content across all use cases: onboarding sequences, promotional campaigns, re-engagement drips, transactional templates, and newsletter issues. It applies proven copywriting formulas (PAS, AIDA, BAB) and adapts tone to match your brand voice.

Beyond single emails, the skill designs multi-email sequences with logical flow, timing gaps, and behavioral triggers. For teams using marketing automation platforms, it outputs sequence logic in a structured format ready for implementation in tools like Klaviyo, HubSpot, or ConvertKit.

The skill also handles subject line generation (with A/B variants), preview text optimization, and CTA refinement to maximize open rates and click-through rates.

## Quick Start

```
/ckm:email --type onboarding --audience trial-users --goal activation
```

## Key Features

- Single email generation for any use case (promo, onboarding, nurture, re-engagement)
- Multi-email sequence design with timing and trigger logic
- Subject line generation with 3-5 A/B test variants
- Preview text optimization paired to each subject line
- Copywriting formulas: PAS, AIDA, BAB, 4Ps
- Brand voice integration via writing style extraction
- Automation flow diagrams (trigger → condition → action)
- Deliverability best practices built in

## Usage Examples

**Onboarding sequence**:
```
Write a 5-email onboarding sequence for new trial users of a project management SaaS.
Goal: get users to create their first project within 7 days.
Tone: friendly, helpful, not pushy
```

**Re-engagement campaign**:
```
Write a 3-email re-engagement sequence for subscribers who haven't opened in 90 days.
Subject line variants for each email. Offer: 20% discount on annual plan
```

**Newsletter issue**:
```
Write this week's newsletter covering: [topic 1], [topic 2], [topic 3].
Audience: B2B marketers. Tone: insightful, concise. Length: 400-500 words
```
