# ckm:social

> Generate platform-optimized social media content and build content calendars for consistent publishing.

## What This Skill Does

The Social skill generates social media content adapted to the conventions, formats, and audiences of each platform. It understands that a LinkedIn thought leadership post, a Twitter thread, an Instagram caption, and a TikTok hook are fundamentally different content types — and writes them accordingly.

Beyond individual posts, the skill builds content calendars: planning what to post, when, and across which platforms. It balances content types (educational, promotional, social proof, engagement), maintains posting frequency, and ties social content to broader campaign themes.

The skill also handles content repurposing — taking a blog post, webinar recording, or customer story and adapting it into multiple social formats without repetitive manual effort.

## Quick Start

```
/ckm:social --platform linkedin --type thought-leadership --topic "AI in marketing"
```

## Key Features

- Platform-native content: LinkedIn, Twitter/X, Instagram, TikTok, YouTube Shorts
- Content type library: educational, promotional, social proof, engagement, trending
- Hashtag research and selection per platform
- Content calendar generation (weekly/monthly)
- Repurposing engine: one source → multiple platform formats
- Brand voice enforcement across all platforms
- Optimal posting time recommendations by platform and audience

## Usage Examples

**LinkedIn post from blog**:
```
Repurpose this blog post into a LinkedIn thought leadership post.
Format: hook → 3 key insights → CTA. Tone: authoritative but approachable. Max 250 words
Blog: [paste blog post]
```

**Weekly content calendar**:
```
Create a weekly social media calendar for next week.
Platforms: LinkedIn, Twitter. 3 posts per platform.
Theme: our new AI analytics feature launch
```

**Twitter thread**:
```
Write a 7-tweet thread on "Why most B2B content fails and what to do instead."
Hook tweet must stop the scroll. End with a CTA to our newsletter
```
