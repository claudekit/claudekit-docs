# `ckm:write`

> Write high-quality marketing content — blog posts, landing page copy, CRO content, and creative pieces.

## What This Skill Does

The Write skill is the primary content generation engine for long-form and creative marketing copy. It produces blog posts, landing page copy, case studies, white papers, product descriptions, and any other written marketing asset — adapting tone, format, and depth to the content type and target audience.

The skill applies conversion-focused writing principles when the goal is action (sign up, buy, request demo) and thought leadership principles when the goal is authority (SEO traffic, newsletter growth, brand building). It uses your brand voice settings and can extract and apply your custom writing style from existing high-performing content.

Write integrates with SEO research, persona profiles, and brand guidelines to produce content that's not just well-written but strategically aligned with marketing goals.

## Quick Start

```
/ckm:write --type blog --topic "How AI is changing B2B marketing" --words 1200 --goal seo
```

## Key Features

- Blog posts, landing pages, case studies, white papers, product descriptions
- Conversion-optimized copy with proven formulas (AIDA, PAS, BAB)
- Brand voice enforcement using style extraction
- SEO integration: keyword targeting, heading structure, meta description
- Persona-targeted writing matched to specific audience needs
- CRO copy: headlines, subheads, CTAs, social proof
- Multiple format outputs: Markdown, HTML, or plain text

## Usage Examples

**SEO blog post**:
```
/ckm:write Write a 1,500-word SEO blog post targeting "email marketing automation for SaaS".
Audience: marketing managers at B2B SaaS companies. Include: 3 main sections, data points,
practical tips. Keyword density: natural, not forced
```

**Landing page copy**:
```
/ckm:write Write the full copy for our free trial landing page.
Product: AI analytics for marketing teams. Audience: CMOs and marketing directors.
Include: hero section, 3 feature blocks, social proof section, FAQ, CTA
```

**Case study**:
```
/ckm:write Write a 800-word customer case study for [Company Name].
Challenge: [X], Solution: our platform, Results: [metrics]. Format: narrative story, not bullet points
```
